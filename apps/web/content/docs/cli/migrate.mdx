## What it does

Applies all pending migrations using dbmate. Migration files live in `./damian/.migrations/`. Only `.sql` files not yet recorded in the migrations table are applied.

---

## Run

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>

```bash tab="npm"
npx damian migrate
```

```bash tab="pnpm"
pnpm damian migrate
```

```bash tab="yarn"
yarn damian migrate
```

</Tabs>
