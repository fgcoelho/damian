## What it does

Reverts the most recent applied migration, returning your database to the previous state.

---

## Run

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>

```bash tab="npm"
npx damian rollback
```

```bash tab="pnpm"
pnpm damian rollback
```

```bash tab="yarn"
yarn damian rollback
```

</Tabs>
