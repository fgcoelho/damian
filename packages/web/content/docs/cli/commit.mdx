## What it does

Commits pending migrations, making them ready to be applied to your production database.

In case there are multiple pending migrations, it will prompt you to select which ones to commit.

## Command

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>
```bash tab="npm"
npm damian commit
```

```bash tab="pnpm"
pnpm damian commit
```

```bash tab="yarn"
yarn damian commit
```
</Tabs>
