## What it does

Generates types based on your migration files in the `./damian/.migrations` folder, outputting them to `./damian/__generated__/schema.ts`.

## Command

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>
```bash tab="npm"
npm damian generate
```
```bash tab="pnpm"
pnpm damian generate
```
```bash tab="yarn"
yarn damian generate
```
</Tabs>
