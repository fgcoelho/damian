## What it does

Runs migrations against your database.

By default, it will apply only committed migrations. Use `--dev` to apply pending migrations as well.

## Command

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>
```bash tab="npm"
npm damian migrate
```
```bash tab="pnpm"
pnpm damian migrate
```
```bash tab="yarn"
yarn damian migrate
```
</Tabs>

## Options 

| Option           | Description                                      |
|------------------|--------------------------------------------------|
| `-d, --dev`     | Allows pending migrations to be applied.           
| `-p, --pick`     | Allows picking specific migrations to apply
