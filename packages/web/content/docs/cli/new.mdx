## What it does

Creates a new pending migration file in the `./damian/.migrations` folder, prompting you for a name.

## Command

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>
```bash tab="npm"
npm damian new
```
```bash tab="pnpm"
pnpm damian new
```
```bash tab="yarn"
yarn damian new
```
</Tabs>
