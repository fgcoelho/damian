## What it does

Undoes the last applied migration, reverting your database to the previous state.

## Command

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>
```bash tab="npm"
npm damian rollback
```
```bash tab="pnpm"
pnpm damian rollback
```
```bash tab="yarn"
yarn damian rollback
```
</Tabs>
