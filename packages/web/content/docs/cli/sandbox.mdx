## What it does

Completely resets your database, generates types, runs all migrations from scratch and apply all sandbox populators.

## Command

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>
```bash tab="npm"
npm damian sandbox
```
```bash tab="pnpm"
pnpm damian sandbox
```
```bash tab="yarn"
yarn damian sandbox
```
</Tabs>
